# ACT 학습 요약 (26.04.20 half data)

## 실행 정보
- 실행 명령: `lerobot-train` (`run_lerobot_train.py` 래퍼 통해 실행)
- 데이터셋: `pipet_26_04_20_half_data`
- 데이터셋 경로: `ai/datasets/26.04.20_half_data`
- 출력 경로: `ai/models/26.04.20_half_data`
- 정책: `act` (ResNet18 backbone)
- 디바이스: `cuda`
- 배치 크기: `8`
- 워커 수: `2`
- 총 스텝: `100000`
- 로그 주기: `50 step`
- 체크포인트 주기: `10000 step`
- Eval 주기: `10000 step`

## 데이터/모델 초기 상태
- `dataset.num_frames=3448`
- `dataset.num_episodes=10`
- `num_learnable_params=34,199,879 (34M)`
- `num_total_params=34,199,879 (34M)`

## 학습 진행 요약
- 초반(수백 step): loss 약 `0.47 -> 0.27`로 빠르게 하락
- 중반(10k~50k): loss 대략 `0.12 ~ 0.06` 구간에서 안정화
- 후반(60k~100k): loss 대략 `0.055 ~ 0.045` 수준 유지, 간헐적 변동
- 최종(100k): 로그 기준 loss 약 `0.044~0.054` 범위
- gradient norm은 초반 고값(30대)에서 후반 1~2대 수준으로 감소

- `60,000`

## 완료 상태
- 학습 진행률: `100% (100000/100000)`
- 총 소요 시간(로그 기준): 약 `45시간 9분 45초`
- 최종 상태: 정상 완료 + 최종 체크포인트 저장 완료

## 메모
- 데이터셋 규모(10 에피소드, 3448 프레임) 대비 학습 스텝이 충분히 큰 편이라 반복 학습이 많이 이루어짐.
- 현재 로그만 보면 학습은 수렴 경향을 보이며 비정상 종료/NaN/폭주 징후는 보이지 않음.
