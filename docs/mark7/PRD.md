---

## 1. 개요

### 제품 설명

Mand.ro Mark7 로봇 핸드를 ROS2 Humble 환경에서 제어하기 위한 지원 라이브러리이다.
실제 하드웨어 제어부터 Gazebo 시뮬레이션, 키보드 조작, 그립 프리셋까지 Mark7 운용에 필요한 기능을 패키지 형태로 제공한다.

### 목표

- Physical AI 연구를 위한 Mark7 조작 플랫폼 구축
- 실제 로봇과 가상 환경을 단일 파라미터로 전환할 수 있는 개발 환경 제공
- Gazebo 시뮬레이션을 통한 직관적인 원격 조작 지원
- 연구실 팀원 누구나 Mark7를 ROS2로 쉽게 사용할 수 있도록 추상화

## 3. 하드웨어 제약

| 항목 | 사양 |
|------|------|
| 로봇 | Mand.ro Mark7 (6 DoF, 와이어 구동) |
| 통신 방식 | 2.4GHz RF 무선 |
| 인터페이스 | USB Serial (Arduino Pro Micro 동글) |
| Baud Rate | 115200 bps (8-N-1) |
| Tx 패킷 | 15 bytes (PC → Hand) |
| Rx 패킷 | 19 bytes (Hand → PC) |
| OS | Ubuntu 22.04 |
| ROS 버전 | ROS2 Humble |
| 시뮬레이터 | Gazebo Classic 11 |

## 4. 기능 요구사항

### 4-1. Driver (핵심)

| ID | 기능 | 설명 | 우선순위 |
|----|------|------|----------|
| F-01 | 실제 로봇 명령 송신 | Tx 동글을 통해 Mark7에 제어 명령 전송 | |
| F-02 | 실제 로봇 상태 수신 | Rx 동글을 통해 position/current/temperature 수신 및 ROS2 토픽 퍼블리시 | |
| F-03 | 가상 모드 | 하드웨어 없이 소프트웨어만으로 동작 확인 | |
| F-04 | 초기화/리셋 | 시작 시 Position Counter 리셋 | |
| F-05 | 안전 모니터링 | 온도/전류 이상 감지 및 경고 | |
| F-06 | 캘리브레이션 | URDF 관절 각도(rad)와 실제 Position 카운터 간 매핑 실측 및 YAML 저장 | |

### 4-2. Gazebo 시뮬레이션

| ID | 기능 | 설명 | 우선순위 |
|----|------|------|----------|
| F-07 | 가상 Mark7 시각화 | Gazebo에서 Mark7 3D 모델 표시 | |
| F-08 | 마우스 조작 | Gazebo GUI에서 마우스로 관절 조작 | |
| F-09 | 실제 로봇 미러링 | 가상 Mark7 동작을 캘리브레이션 매핑 기반으로 실제 Mark7에 전달 | |
| F-10 | RViz 상태 반영 | Rx 수신 데이터를 RViz에 시각화 | |

### 4-3. Teleop

| ID | 기능 | 설명 | 우선순위 |
|----|------|------|----------|
| F-11 | 키보드 조작 | 터미널 키보드로 손가락 실시간 제어 | |
| F-12 | 그립 프리셋 | 자주 쓰는 손 모양을 서비스 콜 하나로 실행 | |

## 5. 비기능 요구사항

| 항목 | 요구사항 |
